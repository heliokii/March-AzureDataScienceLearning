# Learning Data Science from Scratch on Microsoft Azure

This repository provides a structured 31-day roadmap for March 2026 to learn data science (DS) basics using free Microsoft Azure tools. Assumes zero prior knowledge. All resources are free: Microsoft Learn, GitHub, YouTube, and Azure's free tier.

## Prerequisites
- Free Azure account: [azure.microsoft.com/free](https://azure.microsoft.com/free)
- Install Python: [python.org](https://www.python.org)
- VS Code: [code.visualstudio.com](https://code.visualstudio.com)
- Free datasets: [kaggle.com/datasets](https://www.kaggle.com/datasets)

## Roadmap Overview
- **Week 1 (March 1-7):** DS Fundamentals – Python and data basics.
- **Week 2 (March 8-14):** Azure Fundamentals – Cloud setup.
- **Week 3 (March 15-21):** Core DS Skills – Manipulation, viz, intro ML.
- **Week 4 (March 22-28):** Azure-Specific DS – ML Studio and tools.
- **Week 5 (March 29-31):** Projects and Review – Hands-on application.

Navigate to weekly folders for daily tasks. Practice 1-2 hours/day.

## Tips
- Track progress in a personal journal.
- Communities: Reddit r/datascience, r/Azure.
- Extend learning with free Azure certifications on Microsoft Learn.
